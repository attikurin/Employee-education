1. Excel基礎と実践

Excelとは

Microsoft Excelは、表計算ソフトウェアの代表格です。データ入力、計算、分析、グラフ作成など、ビジネスデータを扱う上で欠かせないツールです。

Excelの基本構造

セル（Cell）

データを入力する最小単位。列（A, B, C...）と行（1, 2, 3...）の交点で識別（例：A1セル）

ワークシート（Sheet）

セルが集まった表のこと。1つのファイル（ブック）に複数のシートを作成可能

ブック（Workbook）

Excelファイル全体のこと。拡張子は「.xlsx」

数式（Formula）

「=」から始まる計算式。自動計算により効率的なデータ処理が可能

基本操作

1

データ入力

  • セルをクリックして選択し、キーボードから入力
  • Enterキーで確定→下のセルに移動
  • Tabキーで確定→右のセルに移動
  • 数値、文字列、日付などを自由に入力可能
2

数式の入力

  • 「=」から始める（例：=A1+B1）
  • セルをクリックして参照を簡単に入力
  • Enterキーで計算結果が表示される
例: A1セルに「100」、B1セルに「200」が入力されている場合
C1セルに「=A1+B1」と入力すると → 結果「300」が表示される
3

セルのコピー・貼り付け

  • コピー: Ctrl + C
  • 切り取り: Ctrl + X
  • 貼り付け: Ctrl + V
  • 形式を選択して貼り付け: Ctrl + Alt + V（数式のみ、値のみなど選択可能）
4

オートフィル（連続データの入力）

  • セルの右下の「■」（フィルハンドル）をドラッグ
  • 数値の連番、曜日、月など自動入力
  • 数式を下方向にコピーする際にも便利
例: A1に「1」、A2に「2」と入力し、A1:A2を選択してフィルハンドルをドラッグ
→ 3, 4, 5... と連番が自動入力される
5

行・列の挿入と削除

  • 挿入: 行番号または列番号を右クリック→「挿入」
  • 削除: 行番号または列番号を右クリック→「削除」
  • 複数行・列を選択して一括操作も可能

セル参照の種類

数式をコピーした際の挙動が変わる重要な概念です。

参照の種類 記法 説明 使用例
相対参照 A1 数式をコピーすると、参照先が相対的に移動する（最も一般的） =A1+B1 をコピー → =A2+B2, =A3+B3...
絶対参照 $A$1 数式をコピーしても、参照先が固定される =$A$1*B1 をコピー → =$A$1*B2（A1は固定）
複合参照 $A1 または A$1 列のみ固定（$A1）、または行のみ固定（A$1） =$A1+B1 をコピー → =$A2+C1（列Aは固定、行は移動）

Tips

セル参照に「$」記号を付けるには、数式バーでセル参照部分にカーソルを置き、F4キーを押すと、相対参照→絶対参照→複合参照と順に切り替わります。

2. Excel必須関数

関数とは

Excelの関数は、複雑な計算や処理を簡単に実行できる機能です。ビジネスでよく使う関数を習得することで、作業効率が大幅に向上します。

基本の集計関数

SUM（合計）

用途: 指定した範囲の数値を合計する

書式: =SUM(範囲)

例:
=SUM(A1:A10)
→ A1からA10までの合計を計算
ショートカット: Alt + Shift + =

AVERAGE（平均）

用途: 指定した範囲の平均値を計算

書式: =AVERAGE(範囲)

例:
=AVERAGE(B1:B20)
→ B1からB20までの平均を計算

COUNT（数値の個数）

用途: 数値が入力されているセルの個数をカウント

書式: =COUNT(範囲)

例:
=COUNT(C1:C50)
→ C1からC50で数値が入っているセルの個数
COUNTA: 空白以外のセルをカウント

MAX / MIN（最大・最小）

用途: 最大値・最小値を取得

書式: =MAX(範囲) / =MIN(範囲)

例:
=MAX(D1:D100) → 最大値
=MIN(D1:D100) → 最小値

論理関数

IF関数（条件分岐）

用途: 条件によって処理を分ける

書式: =IF(条件, 真の場合の値, 偽の場合の値)

例1: 基本的な使い方 
=IF(A1>=80, "合格", "不合格")

→ A1が80以上なら「合格」、それ以外なら「不合格」

例2: 複数条件（IF関数のネスト） 
=IF(A1>=90, "優", IF(A1>=80, "良", IF(A1>=70, "可", "不可")))

→ 90以上:優、80以上:良、70以上:可、それ以外:不可

例3: 空白セルの判定 
=IF(A1="", "未入力", A1)

→ A1が空白なら「未入力」、それ以外はA1の値

AND / OR関数（複数条件）

用途: 複数の条件を組み合わせる

AND関数（すべての条件が真） 
=IF(AND(A1>=70, B1>=70), "合格", "不合格")

→ A1とB1の両方が70以上なら「合格」

OR関数（いずれかの条件が真） 
=IF(OR(A1="完了", A1="済"), "OK", "未完了")

→ A1が「完了」または「済」なら「OK」

検索・参照関数

VLOOKUP関数（縦方向検索）

用途: 別の表から該当するデータを探して取得する（ビジネスで超頻出！）

書式: =VLOOKUP(検索値, 範囲, 列番号, 検索方法)

引数の説明
  • 検索値: 探したい値（例：社員番号、商品コード）
  • 範囲: データが入っている表全体（1列目に検索値がある必要あり）
  • 列番号: 取得したい列が範囲の何列目かを指定（1から始まる）
  • 検索方法: FALSE（完全一致）またはTRUE（近似値）※通常はFALSE
実践例

シナリオ: 社員番号から社員名を取得したい

【社員マスタ（A1:C5）】
A列: 社員番号 B列: 社員名 C列: 部署
1001山田太郎営業部
1002佐藤花子企画部
1003鈴木一郎総務部
1004田中美咲営業部
 
=VLOOKUP(1002, A2:C5, 2, FALSE)

→ 結果: 「佐藤花子」

説明: 社員番号「1002」をA2:C5の範囲の1列目（A列）から探し、2列目（B列）の値を取得

VLOOKUP使用時の注意点
  • 検索値は範囲の1列目になければならない
  • 検索方法はFALSE（完全一致）を推奨（TRUEは昇順ソート必須）
  • 範囲は絶対参照（$A$2:$C$5）にすると数式コピーが楽
  • 検索値が見つからない場合は#N/Aエラーが表示される

XLOOKUP関数（最新の検索関数）

用途: VLOOKUPの進化版。より柔軟な検索が可能（Microsoft 365、Excel 2021以降）

書式: =XLOOKUP(検索値, 検索範囲, 戻り範囲)

VLOOKUPとの違い
  • 列番号の指定が不要（戻り範囲を直接指定）
  • 検索値が左端になくてもOK
  • 右から左への検索も可能
  • 検索値が見つからない場合の代替値を指定可能
=XLOOKUP(1002, A2:A5, B2:B5, "見つかりません")

→ 社員番号1002をA2:A5から探し、対応するB2:B5の値を取得。見つからなければ「見つかりません」を表示

INDEX + MATCH関数（高度な検索）

用途: VLOOKUPより柔軟な検索（左方向検索、複数条件検索が可能）

書式の組み合わせ: 
=INDEX(取得したい列, MATCH(検索値, 検索列, 0))
実例: 
=INDEX(B2:B5, MATCH(1002, A2:A5, 0))

→ A2:A5から「1002」を探し、その位置に対応するB2:B5の値を取得

VLOOKUPより優れている点:
  • 検索列が右端でも左端でもOK
  • 列を追加・削除しても数式が壊れにくい
  • より高速に動作する（大量データの場合）

文字列操作関数

CONCATENATE / &

用途: 複数のセルの文字列を結合

=A1&" "&B1
または
=CONCATENATE(A1, " ", B1)
→ A1「山田」、B1「太郎」なら「山田 太郎」

LEFT / RIGHT / MID

用途: 文字列の一部を取り出す

=LEFT(A1, 3) → 左から3文字
=RIGHT(A1, 2) → 右から2文字
=MID(A1, 2, 3) → 2文字目から3文字

LEN（文字数）

用途: 文字列の文字数を取得

=LEN(A1)
→ A1「こんにちは」なら結果「5」

TRIM（空白削除）

用途: 余分な空白を削除

=TRIM(A1)
→ 「 山田 太郎 」→「山田 太郎」

日付・時刻関数

TODAY / NOW

用途: 今日の日付・現在時刻を自動取得

=TODAY() → 今日の日付
=NOW() → 現在の日時
ファイルを開くたびに自動更新される

DATE / YEAR / MONTH / DAY

用途: 日付の作成・分解

=DATE(2025, 3, 20) → 2025/3/20
=YEAR(A1) → 年を取得
=MONTH(A1) → 月を取得
=DAY(A1) → 日を取得

DATEDIF（日数計算）

用途: 2つの日付間の日数・月数・年数を計算

=DATEDIF(A1, B1, "D") → 日数
=DATEDIF(A1, B1, "M") → 月数
=DATEDIF(A1, B1, "Y") → 年数

WEEKDAY（曜日）

用途: 日付から曜日を取得（1=日曜、7=土曜）

=WEEKDAY(A1)
土日判定:
=IF(OR(WEEKDAY(A1)=1, WEEKDAY(A1)=7), "休日", "平日")

3. グラフ・ピボットテーブル

Excelグラフの作成

データを視覚化することで、傾向やパターンを一目で把握できます。

主要なグラフの種類と用途

縦棒グラフ

用途: 項目間の比較

例: 部門別売上、商品別販売数

折れ線グラフ

用途: 時系列の推移

例: 月別売上推移、株価の変動

円グラフ

用途: 構成比率の表示

例: 市場シェア、予算配分

面グラフ

用途: 累積推移の可視化

例: 積み上げ売上推移

散布図

用途: 2変数の相関関係

例: 広告費と売上の関係

ヒートマップ

用途: 数値の大小を色で表現

例: 時間帯別アクセス数

グラフ作成の基本手順

STEP 1
データ範囲を選択

グラフにしたいデータ（項目名とデータ）を選択

STEP 2
グラフを挿入

「挿入」タブ → 「グラフ」グループから適切なグラフを選択

STEP 3
グラフをカスタマイズ

タイトル、軸ラベル、色、凡例などを編集

STEP 4
完成

グラフをコピーして他の資料（PowerPointなど）に貼り付け可能

グラフ作成のポイント

  • グラフタイトルは具体的に（「売上」→「2024年度 部門別月次売上推移」）
  • 軸ラベルを必ず表示（単位も記載: 「売上（百万円）」）
  • 色は3～5色程度に抑える（多すぎると見づらい）
  • 凡例の位置を工夫（グラフの邪魔にならない場所に配置）
  • データラベルは必要に応じて表示（全項目に付けると煩雑）

ピボットテーブル（データ集計の最強ツール）

ピボットテーブルは、大量のデータを瞬時に集計・分析できる強力な機能です。ドラッグ&ドロップだけで、様々な角度からデータを見ることができます。

ピボットテーブルでできること

集計

合計、平均、個数などを自動計算

グループ化

月別、部門別など自由にグループ化

フィルタリング

必要なデータだけを表示

並べ替え

昇順・降順で簡単にソート

ピボットテーブルの作成手順

1
データ範囲内のセルを選択

見出し行を含むデータ範囲のいずれかのセルをクリック

2
ピボットテーブルを挿入

「挿入」タブ → 「ピボットテーブル」をクリック

3
配置場所を選択

「新規ワークシート」または「既存のワークシート」を選択

4
フィールドを配置

右側のフィールドリストから、以下のエリアにドラッグ：

  • 行: 縦方向に表示する項目
  • 列: 横方向に表示する項目
  • 値: 集計する数値
  • フィルター: データの絞り込み条件

実践例：売上データの分析

元データ（売上明細）:

日付 商品 担当者 売上金額
2025/1/5商品A山田50,000
2025/1/8商品B佐藤30,000
2025/2/3商品A山田45,000
... （1000件のデータ）

ピボットテーブルでの集計結果例:

  • 行: 商品、列: 月、値: 売上金額の合計
    → 商品別・月別の売上一覧表が瞬時に作成される
  • 行: 担当者、値: 売上金額の合計
    → 担当者別の売上ランキングが作成される

ピボットテーブル活用Tips

  • データ更新後は「データ」タブ → 「更新」をクリックして最新化
  • 「値フィールドの設定」で集計方法を変更可能（合計、平均、個数など）
  • 日付は自動的に月、四半期、年にグループ化できる
  • 計算フィールドで独自の計算式を追加可能（例：利益率 = 利益 ÷ 売上）
  • ピボットグラフと組み合わせると、より視覚的な分析が可能

4. PowerPoint基礎

PowerPointとは

Microsoft PowerPointは、プレゼンテーション資料作成の定番ソフトウェアです。スライドを作成し、ビジュアルを活用して情報を効果的に伝えることができます。

PowerPointの基本構造

スライド（Slide）

プレゼンテーションの1ページ。テキスト、画像、グラフなどを配置

レイアウト（Layout）

スライドの構成テンプレート。タイトルスライド、コンテンツスライドなど

テーマ（Theme）

色、フォント、背景のデザインセット。全スライドに統一感を与える

スライドマスター（Master）

全スライドに共通する要素（ロゴ、フッターなど）を一括管理

スライド作成の基本原則

1. シンプルに（Keep It Simple）

1スライド1メッセージが基本。情報を詰め込みすぎない。

❌ 悪い例: 小さな文字で長文がびっしり、複数のグラフが並ぶ
✅ 良い例: キーワードのみ大きく表示、1つのグラフを強調

2. 視覚的に（Visual First）

文字だけでなく、画像、アイコン、グラフを活用する。

効果: 視覚情報は文字情報の6万倍速く処理される（3M Corporation調査）

3. 一貫性（Consistency）

フォント、色、レイアウトを統一する。

推奨: フォントは2種類まで、色は3～4色まで

4. ストーリー性（Story Flow）

起承転結を意識した構成。聞き手を飽きさせない流れを作る。

基本構成: 導入 → 現状・課題 → 解決策 → 効果・まとめ

5. 読みやすさ（Readability）

フォントサイズは最小18pt以上（推奨24pt以上）。

カラーコントラスト: 背景と文字の色のコントラストを強く（白背景なら濃い色の文字）

基本的なスライドレイアウト

1. タイトルスライド

プレゼンテーションのタイトル
副題・日付・発表者名

用途: プレゼンの冒頭、セクション区切り

2. タイトル+コンテンツ

スライドタイトル
• ポイント1
• ポイント2
• ポイント3

用途: 最も一般的なレイアウト

3. 2カラム（左右分割）

比較・対比
左側の内容
右側の内容

用途: Before/After、メリット/デメリットの比較

4. 画像+説明

ビジュアル重視
画像
説明テキスト

用途: 製品紹介、事例紹介

5. 資料デザインの基本

配色の原則

60-30-10の法則

プレゼンデザインの黄金比率

ベースカラー 60%

背景色、主要な色

メインカラー 30%

見出し、重要部分

アクセントカラー 10%

強調、CTA

おすすめ配色パターン

ビジネス: ネイビー + ブルー + ライトグレー（信頼感）

エコ・環境: グリーン系（自然、成長）

活力・情熱: オレンジ系（エネルギッシュ）

創造性・高級感: パープル系（洗練）

配色のコツ

  • 会社のコーポレートカラーを基本に
  • 3色以内に抑える（多すぎると散漫な印象）
  • 背景と文字のコントラストを十分に（白背景なら濃い色の文字）
  • カラーパレットツール活用（Adobe Color、Coolorsなど）

フォントの選び方

フォントの種類

種類 特徴 用途 例
ゴシック体 太さが均一、読みやすい 見出し、本文 メイリオ、游ゴシック
明朝体 格調高い、伝統的 フォーマルな資料 游明朝、MS明朝
Sans-serif シンプル、モダン 英数字 Arial, Calibri

推奨フォントサイズ

  • タイトル: 36～44pt（目立たせる）
  • 見出し: 28～32pt（階層を明確に）
  • 本文: 20～24pt（読みやすく）
  • 注釈・出典: 14～16pt（小さすぎ注意）

フォント選びのポイント

  • フォントは2種類まで（見出しと本文で使い分け）
  • スライドタイトルは太字（Bold）で強調
  • 行間を適度に空ける（1.2～1.5倍が目安）
  • 文字の詰め込みすぎに注意（余白を大切に）

レイアウトの原則

1. 整列（Alignment）

要素を揃えることで秩序と統一感が生まれる

✅ 良い例: すべての要素が左揃え、または中央揃えで統一
❌ 悪い例: 要素がバラバラに配置されている

2. 近接（Proximity）

関連する要素を近づけ、関連しない要素は離す

効果: 情報のグループ化が一目で分かる

3. 余白（White Space）

適度な余白が洗練された印象を与える

目安: スライドの30～40%は余白にする

4. コントラスト（Contrast）

重要な情報を目立たせる

方法: サイズ、色、太さで差をつける

5. 反復（Repetition）

同じデザイン要素を繰り返し使用

効果: 全体の統一感、ブランディング

6. 実践テクニック集

Excel: データの入力規則

方法: 「データ」タブ → 「データの入力規則」

Excel: テーブル機能

よくあるミスと対策

Excel: 数式がコピーできない

原因: セルが文字列形式になっている

対策: セルの表示形式を「標準」または「数値」に変更

Excel: #N/Aエラー

原因: VLOOKUPで検索値が見つからない

対策: IFERROR関数で囲む → =IFERROR(VLOOKUP(...), "該当なし")

PowerPoint: フォントが変わってしまう

原因: 他のPCで開いたときにフォントがない

対策: 「ファイル」→「オプション」→「フォントの埋め込み」をチェック

PowerPoint: ファイルサイズが大きすぎる

原因: 画像が圧縮されていない

対策: 画像を選択 → 「図の圧縮」 → 解像度を下げる
