
### 方法2: 簡易版作成

現在、各セクションは完全に独立しているので、sections/フォルダ内の各HTMLファイルを個別に表示することも可能です。

#### 各セクション個別表示用のHTMLを作成する場合:
各セクションファイルに以下のテンプレートを追加します：

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>セクション名 - 社員教育ガイド</title>
    <link rel="stylesheet" href="../css/style.css">
    <link rel="stylesheet" href="../css/additional-styles.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@300;400;500;700;900&display=swap" rel="stylesheet">
</head>
<body>
    <!-- セクションの内容 -->
    
    <script src="../js/main.js"></script>
</body>
</html>
```

## 📞 サポート

### よくある質問

**Q: セクションの順番を変更できますか？**
A: はい。HTMLのセクションタグの順番を入れ替えるだけで変更できます。ナビゲーションメニューの順番も合わせて変更してください。

**Q: 自社用にカスタマイズしたいです**
A: CSS variablesを変更することで、カラースキームを簡単に変更できます。style.cssの`:root`セクションを編集してください。

**Q: 印刷時にセクションを非表示にできますか？**
A: CSSの`@media print`セクションに非表示にしたいセクションのスタイルを追加してください。
